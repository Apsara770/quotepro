# 🎨 QuotePro – Premium Web-Based Graphic Design Tool


## 🚀 Overview

**QuotePro** is a modern, browser-based graphic design application built using **HTML, CSS, and JavaScript**.
It allows users to create high-quality social media posts, quotes, and digital designs with a **simple yet powerful interface**.

👉 Designed as a **mini Canva-style editor** with real-time editing and export features.

---

## ✨ Features

### 🖼️ Canvas & Layout

* Multiple aspect ratios:

  * 1:1 (Instagram Post)
  * 9:16 (Stories / Reels)
  * 16:9 (YouTube Thumbnail)
* Centered design workspace

---

### 🔤 Typography

* Multiple fonts (Arial, Roboto, Montserrat, Pacifico)
* Adjustable:

  * Font size
  * Color
  * Bold / Italic styles
* Live preview updates instantly

---

### 🧩 Multi-Element Editing

* Add multiple text elements
* Click to edit text
* Drag & position anywhere
* Real-time UI updates

---

### 🖼️ Image Features

* Upload background images
* Add logos or personal photos
* Drag & resize images
* Apply shapes:

  * Circle
  * Oval
  * Square

---

### 🌍 Stock Images

* Integrated with Unsplash API
* Search and import high-quality images directly

---

### 🔄 Editing Experience

* Smooth interactions
* Smart positioning system
* Clean and intuitive controls

---

### 📤 Export Options

* Download as **PNG** using html2canvas
* Download as **PDF** using html2pdf.js

---
